# apache-beam-spanner
Repository to host prototype codes for Apache Beam and Cloud Spanner integration

Pipelines Code in this repository :

1. Read from Spanner in dataflow and write to Spanner and GCS
2. Read from Cloud SQL and Write to Spanner table and GCS in Avro format
3. Read from PubSub and Write to BigQuery
4. Read from PubSub and Write to Spanner
